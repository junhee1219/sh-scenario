  • 2026.7.22 ~ 7.24

    계약 체결 + 계약금 10%(8,190만원) 납부. 순수현금으로 충당. 전자계약(irts.molit.go.kr)으로 하면 전세대출 금리 0.1~0.2%p 할인.

  • 2026.7 ~ 8

  • 2026.8.24 ~ 10.23

    입주지정기간 = 잔금 90%(7.37억) 납부 · 입주 · 전입신고. 낼 수 있는 만큼 먼저 내고 부족분은 아래 지연 활용.

  • ~ 2027.1.23

  • 2027.5

    기존전세(A) 만기 → 보증금 4.3억 반환 → 대출 3억 상환 → 순현금 1.3억 확보. 역전세 없어 반환은 확정.

  • ~ 2028 (2년 후)

    첫 재계약. 이 전에 둘째 출산(입주 후)이면 소득·자산 초과 면제 → 20년 거주 안정.

04 은행 전략

전세대출은 은행마다 한도·금리·소득 인정이 크게 달라 — 반드시 여러 곳 비교.

  • 3~4곳 사전상담(가심사) 후 비교. 주거래은행 1곳 + SH 장기전세 협약은행 2~3곳. 가심사 여러 번 돌려도 신용점수 영향 거의 없음.
  • 주거래은행은 꼭 포함 — 급여이체·카드실적 우대금리(0.1~0.3%p). 단 주거래만 보면 한도 손해 볼 수 있음.
  • SH 협약은행 확인 — 공공(SH)이 임대인이라 전용 상품·우대 가능. SH 콜센터 1600-3456 또는 계약서류에서 목록 확인.
  • "아파트 지점"은 무관 — 전세대출은 어느 지점서든 가능(분양 집단대출과 다름).
소득증빙 — 작년치에 갇히지 말 것. 이직 등으로 작년 원천징수 소득이 낮게 잡히면, 재직증명서 + 최근 급여명세서 + 건강보험 보수월액으로 현 소득 연환산을 인정받자. 배우자 소득은 복직예정확인서 등으로 합산 시도 → 한도 상승.

은행에 반드시 물어볼 것

  • 이 건(SH 장기전세 8.19억 아파트) 전세대출 최대 한도는? (5억 되면 받아두면 주식 방어에 유리)
  • 현 소득 연환산 인정 되나? 배우자 소득 합산 조건은?
  • 메이플자이 등기 완료 상태라 대출 실행 가능한가?
  • 우대금리 다 적용 시 최종 금리는? (전자계약 0.1~0.2 + 신혼/다자녀 + 주거래 + SH협약 → 0.3~0.7%p 감면 가능)
  • 곧 시행될 고가전세·무주택 DSR 규제가 이 건에 걸리나?
  • 브릿지용 신용대출 한도(DSR 40% 내)는 얼마?

05 집주인(기존전세 A) 협상

만기 2027.5이지만 조기반환을 끌어내면 브릿지가 통째로 사라진다.

  • 안전판 먼저: 만기 6~2개월 전(2026.11~2027.3) 갱신거절(퇴거) 의사 문서 통보 → 만기 반환은 법적 의무로 확정.
  • 조기반환 지렛대: 전세난 + 매매가 10억↑ → 내놓으면 바로 나가고, 집주인은 보증금 올려받을 기회라 거절 명분 적음.
  • 협조 유도: "새 세입자 구해지면 나가겠다" + 중개수수료 내가 부담 제안 → 집주인 부담 0.
협상 멘트 예시
“사정상 만기(2027.5)보다 앞당겨 이사해야 할 것 같습니다. 요즘 전세 물건이 귀하니 새 세입자는 제가 부동산 통해 빨리 맞추고, 중개수수료도 제가 부담하겠습니다. 새 임차인 계약되는 시점에 보증금 반환해 주시면 됩니다.”

받을 수 있는 지원을 전부 대조했다. 정책 초저금리는 8.19억 보증금 벽에 다 막히고, 실익은 두 곳에 있다.

08 20년 거주 유지 메모 (재계약)

  • 재계약 소득기준 (49㎡=50㎡미만): 105% / 맞벌이 140% = 월 1,067만원(3인 기준). 소득이 이 선에 근접하면 초과 위험 — 상여·복직 등으로 오를 수 있음.
  • 둘째(입주 후 출생) → 소득·자산 초과 면제. 첫 재계약(≈2028) 전 출산이 핵심. 20년 안정의 열쇠.
  • 단, 자동차가액 4,563만원 이하는 면제 안 됨 — 비싼 차 사면 재계약 불가.
  • 총자산 6.4억 이하 유지 (해당 장기전세 보증금은 산정 제외가 원칙, SH 재확인).
  • 무주택 · 전입 · 실거주 임대기간 내내 필수. 전대·전매 금지(위반 시 형사처벌).

09 액션 플랜

시점별 체크리스트 — 위에서 아래로.

이번 주 (D-14, ~7.24 전)

  • 은행 3~4곳 사전상담 — SH 협약은행 목록부터 확보(콜센터 1600-3456). 최대 한도·최종 금리 비교.
  • 메이플자이 등기 완료 여부 확인 — 대출 실행 가능한지(등기 지연 시 대출 불가 리스크).
  • 소득증빙 서류 준비 — 재직증명서 · 급여명세서 3~6개월 · 건강보험 보수월액 · 배우자 복직예정확인서(해당 시).
  • 계약금 8,190만원 재원(순수현금) 확보.

계약 (2026.7.22 ~ 7.24)

  • 계약금 납부 후 전자계약 체결 → 금리할인 확보.
  • 공사에 연체료율·3개월 지연 규정 · 부분납부 처리 방식 문서로 재확인.

입주 전 (2026.8 ~ 10, 지연 시 ~2027.1)

  • 기존전세 A 집주인 조기반환 협상 — 새 세입자 조건 + 중개수수료 부담 제안. (성사되면 길 A)
  • 협조 불투명하면 주식담보대출 · 신용대출 한도 미리 확보(길 B 대비).
  • 전세대출 실행 → 잔금 납부 → 입주 + 전입신고(대항력·자격 유지).

장기 (2027 ~)

  • 2027.5 A 반환 → 브릿지/신용대출 상환.
  • 첫 재계약(≈2028) 전 둘째 출산(입주 후) → 소득·자산 면제 확보.
  • 총자산 6.4억 · 자동차 4,563만원 기준 관리.
